This program recreats a PVP tank battle game, Battle City, on FPGA. 

To play the game, you need to:
	1. Complie the program.
	2. Connect to a FPGA board.
	3. Connect to a monitor via VGA.
	3. Program the Device.
	4. Run NIOS II. 

Player1: 
	W
      A S D
attack: J

Player2:
	up
     < down >
attack: 1

Pause:
	ESC
